# data/

ここには **生成されたデータ**（CSV/JSONL）を置きます。

- リポジトリにはサイズ控えめの `sample/` だけ入れています（動作確認用）。
- フルサイズを使う場合は `src/make_dataset.py` で手元生成する想定です。

## カラムざっくり

- `u`：銀河帯のどの位置を見てるか（中心に近いほど“厚い”想定）
- `overlap`：レンズの重なり（鮮明さ）
- `star_count`：見える星の数（カウント）
- `haze_obs`：ぼんやり白さ（連続値）
- `memo`：本文由来の観測メモ（雰囲気担当）

### おまけ（ground truth）
生成時の“真値”列（チェック用）。公開用の体験としては、隠したまま遊ぶのもアリです。

- `thickness_true`：レンズの厚み（先生の模型のイメージ）
- `true_lambda`：星数の期待値（Poissonのλ）
- `noise_sigma`：白さの観測ノイズ量
- `overdispersed`：一部だけ過分散を混ぜた印（議論のタネ）

### 仕掛け（event）
v2では、外れ値っぽい1点を混ぜるオプションがあります（“観測の断絶”のタネ）。
- `event_flag`, `event_name`
